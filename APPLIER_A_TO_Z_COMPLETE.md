## 💻 Quick Start

```bash
# Install
pip install anthropic playwright requests google-auth-oauthlib google-api-python-client
playwright install
export ANTHROPIC_API_KEY="sk-ant-..."

# Test New Features
./applier-pro help
./applier-pro followup schedule --company "Google" --job-title "SWE" \
                                --email recruiter@google.com --name "Jane" \
                                --type interview_thank_you
./applier-pro calendar create --company "Anthropic" --job-title "SWE" \
                              --datetime "2025-12-25 14:00"
./applier-pro linkedin --job-url [LinkedIn URL]
```
