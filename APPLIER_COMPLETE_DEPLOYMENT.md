## 💡 How to Use

### For You (Alexa)

**1. Set Up CLI (5 minutes):**
```bash
cd ~/blackroad-sandbox

# Install dependencies
pip install anthropic playwright requests
playwright install

# Set API key
export ANTHROPIC_API_KEY="sk-ant-..."

# Test it
./applier-pro help
./test-applier-pro.sh
```

**2. Generate Your First Cover Letter (5 minutes):**
```bash
# Create a test job description
echo "Senior Software Engineer at Google..." > job.txt

# Generate cover letter
./applier-pro cover --job-title "Senior SWE" \
                     --company "Google" \
                     --job-description job.txt \
                     --variants 2

# Review both variants in ~/.applier/cover_letters/
```

**3. Start Applying (Daily):**
```bash
# Search for jobs
./applier-real search

# Batch apply
./applier-pro batch --max 10 --min-score 75
```

**4. Prepare for Interviews:**
```bash
# Research company
./applier-pro research "Anthropic"

# Practice interview
./applier-pro interview
```

**5. Negotiate Offers:**
```bash
# Analyze and negotiate
./applier-pro salary
```
