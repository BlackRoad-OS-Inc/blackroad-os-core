## 2. q as the Scaling / Deformation Dial

### Classical q-Objects

**Credit:** Classical mathematics (q-calculus, basic hypergeometric series)

**q-Pochhammer Symbol:**
```
(a;q)_n = ∏_{k=0}^{n-1} (1 - a·q^k)
```

**q-Factorial:**
```
[n]_q! = ∏_{k=1}^{n} (1 - q^k)/(1 - q)
```

**q-Number:**
```
[n]_q = (1 - q^n)/(1 - q)
```

### Amundson Interpretation

**q encodes how "non-classical" or "non-linear" the accumulation is.**

- q → 1: Classical/linear regime
- q ≠ 1: Deformed, biased, or "quantum-ish" regime

**In BlackRoad terms:**
q turns the world from "boring add/multiply" into "bias toward certain paths/futures/weights."
